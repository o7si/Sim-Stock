**简要描述：**

- 获取股票列表。

**请求URL：**

- ` /Sim-Stock/stock/getList `

**请求方式：**

- GET

**参数：**

| 参数名 | 类型 | 默认值 | 说明 |
| --- | --- | :-: | --- |
| pageNumber | int | 1 | 页码 |
| pageSize | int | 5 | 每页记录条数 |

**请求示例**

` /Sim-Stock/stock/getList?pageNumber=1&pageSize=3 `
