**简要描述：**

- 获取股票变化信息。

**请求URL：**

- ` /Sim-Stock/market/getLateList `

**请求方式：**

- GET | POST

**参数：**

| 参数名 | 类型 | 说明 |
| --- | --- | --- |
| stockId | int | 股票编号 |
| recordNumber | int | 信息条数 |

**请求示例**

```
{
	"stockId": 1,
	"recordNumber": 2
}
```
